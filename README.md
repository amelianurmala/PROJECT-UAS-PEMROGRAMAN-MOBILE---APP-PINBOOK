## Deskripsi Aplikasi PinBook
PinBook adalah aplikasi perpustakaan berbasis Android yang dibuat untuk membantu proses pengelolaan dan peminjaman buku secara digital. Aplikasi ini memudahkan pengguna dalam mengakses informasi buku, melakukan peminjaman, serta melihat riwayat peminjaman tanpa harus menggunakan pencatatan manual.

Pada saat aplikasi dijalankan, PinBook menampilkan splash screen yang dilengkapi dengan fitur deteksi lokasi pengguna. Berdasarkan lokasi tersebut, aplikasi menampilkan sapaan “Say Hello” sesuai daerah serta logo atau bendera daerah setempat, sesuai dengan ketentuan pada project Ujian Akhir Semester Pemrograman Mobile 1.

## Fitur Aplikasi
- Splash Screen
  - Deteksi lokasi pengguna
  - Menampilkan logo/bendera daerah
  - Say Hello sesuai lokasi
- Login & Register
- Halaman utama aplikasi
- UI & UX sederhana

## Teknologi
- Android Studio
- Kotlin
- XML Layout
